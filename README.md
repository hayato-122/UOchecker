魚の判別、AIの利用やAPIを使用するところからAIに強く、リクエスト、レスポンスの処理が単純であるPythonを採用しました。
漁業権の情報取得には海洋状況表示システム(海しる)を使用し、マップで選択された位置の緯度経度を送信を行い、座標の漁業権情報を取得しています。
Gemini、Chatgptなどと比較してレスポンスが早い点や、事前学習によって学習コストが低いことからClaudeを選択しました。
Claudeではアップロードされた画像の魚種の識別を行い、海しるの情報と識別した魚種を比較することで、漁業権で保護されている魚かどうかを判別しています。
サーバーにはDockerとの組み合わせや無料で使用できるサーバーの中でCPU、メモリ量がAWSなどの他のサービスより優れているため、より多くのアクセスに耐えられる点からHuggingFaceを使用しています。
